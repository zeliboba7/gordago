Gordgao IOTicks 1.0
www.gordago.com
------------------------------------------------------------------------

IOTicks - программа для импорта тиковой истории от компании Gain Capital для программы Forex Optimizer TT

Скачать файлы тиковой истории компании Gain Capital можно здесь: http://ratedata.gaincapital.com

Например, необходимо добавить тиковую историю за 2002 год по паре EURUSD.
1) Качаем тиковую историю с сайта Gain Capital: http://ratedata.gaincapital.com/2002/EUR_USD_2002.zip
и разархивируем в папку программы IOTicks\Import\CainCapital
2) Копируем файл истории ForexOptimizer по EURUSD (по умолчанию c:\Program Files\Gordago\Forex Optimizer TT\history\EURUSD.gtk)
в папку программы IOTicks\data\history
3) Запускаем программу и жмем кнопку Import.
после того как строка по валюте EURUSD станет белым цветом - история загружена. 
4) Копируем файл EURUSD.gtk из IOTicks\data\history в папку программы Forex Optimizer TT\history
